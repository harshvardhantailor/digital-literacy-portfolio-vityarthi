# Task 1: Digital Literacy Awareness Infographic

In this task, I created a digital literacy awareness infographic using Canva. The infographic is designed to help students understand the concept of digital literacy and its importance in today's digital world.

The design covers key topics such as the definition of digital literacy, safe internet practices, useful digital tools for students, professional online presence, email etiquette, and responsible digital behavior. Each section is presented in a clear and structured format using bullet points and visual elements.

The infographic aims to provide simple and easy-to-understand information so that students can learn how to use digital technologies effectively, safely, and responsibly.

The final infographic has been exported as a PDF file and is included in this folder.
